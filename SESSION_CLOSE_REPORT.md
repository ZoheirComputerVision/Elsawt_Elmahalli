### قرارات تقنية تم اتخاذها (يجب ألا تتغير)

#### قرارات الجلسة الحالية (إصلاح الصورة)
- الصور تُحفظ كملفات في `public/uploads/` بدلاً من Base64 في JSON
- المسار `/uploads/img_xxx.jpg` يُخزن في `image_data` بدل Base64
- حد أقصى للصورة: **5 MB** (فحص عميل)
- مهلة للطلب: **30 ثانية** (AbortController)
- أخطاء قاعدة البيانات تنتشر إلى المستخدم (لا تُبلع)

#### قرارات سابقة
- النشر: Git commit → Git push → GitHub → HostingGuru.io (auto-deploy)
- **🚫 لا تُسأل المستخدم**: نفّذ `git push` مباشرة بعد أي commit
- القائمة المنسدلة توضع **بعد "الرئيسية" وقبل "الأخبار"**
- الروابط الخارجية تفتح في نافذة جديدة (`target="_blank"`)
- الموقع RTL بالعربية بالكامل

## 💾 4. سياق المشروع (Project Context)

### ملخص المشروع
نظام "الجريدة المدرسية الذكية" منصة آلية بالذكاء الاصطناعي لثانوية المجاهد خليل محمد (عين كرمس - تيارت، الجزائر). يجمع المحتوى من Facebook وصفحة وزارة التربية الوطنية، يحلله ويصنفه عبر AI، ثم ينشره تلقائياً بعد مراجعة بشرية. واجهة عامة + أرشيف زمني + لوحة تحكم كاملة.

### الـ Stack التقني
| الطبقة | التقنية |
|--------|---------|
| **الخادم** | Node.js + Express |
| **قاعدة البيانات** | SQLite + JSON files للتخزين (جدول لكل كيان في ملف منفصل) |
| **الواجهة** | HTML5 + CSS3 (Vanilla) + JavaScript (Vanilla) |
| **الأمان** | Helmet.js + CORS |
