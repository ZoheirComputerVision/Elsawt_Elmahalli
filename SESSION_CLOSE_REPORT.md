## 💾 4. سياق المشروع (Project Context)

### ملخص المشروع
نظام "الجريدة المدرسية الذكية" هو منصة آلية بالذكاء الاصطناعي لثانوية المجاهد خليل محمد (عين كرمس - تيارت، الجزائر). يقوم بجمع المحتوى من Facebook ووزارة التربية الوطنية، تحليله وتصنيفه عبر AI، ثم نشره تلقائياً مع مراجعة بشرية. يحتوي على واجهة عامة، أرشيف زمني، ولوحة تحكم إدارية كاملة.

### الـ Stack التقني
| الطبقة | التقنية |
|--------|---------|
| **الخادم** | Node.js + Express |
| **قاعدة البيانات** | SQLite + JSON files للـ cache |
| **الواجهة** | HTML5 + CSS3 (Vanilla) + JavaScript (Vanilla) |
| **الأمان** | Helmet.js + CORS |
| **الجدولة** | node-cron (جمع/تحليل/نشر كل 30-15-10 دقيقة) |
| **النشر** | GitHub → HostingGuru.io (deploy تلقائي) |

### مشاريع أخرى في نفس المسار

| المشروع | التقنية | الحالة |
|---------|---------|--------|
| `q-flow/` | Next.js 16 + Prisma + Tailwind 4 | قيد التطوير (Micro SaaS لإدارة الطوابير) |
| `SmartBrowserTracker/` | .NET 10 WPF + EF Core + SQLite | مكتمل (تعقب المتصفح المحلي) |

### ملاحظات وقيود
- الموقع **RTL بالعربية** بالكامل
- **Vanilla JS** — لا React/Vue/Angular
- الصور تُخزّن كـ **Base64** في SQLite
- بيئة التطوير: **Windows** — بيئة الإنتاج: **Linux (HostingGuru.io)**
