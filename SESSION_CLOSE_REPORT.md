## 💾 4. سياق المشروع (Project Context)

### ملخص المشروع
نظام "الجريدة المدرسية الذكية" هو منصة آلية بالذكاء الاصطناعي لثانوية المجاهد خليل محمد (عين كرمس - تيارت، الجزائر). يقوم بجمع المحتوى من Facebook ووزارة التربية الوطنية، تحليله وتصنيفه عبر AI، ثم نشره تلقائياً مع مراجعة بشرية اختيارية. يحتوي على واجهة عامة، أرشيف زمني، ولوحة تحكم إدارية كاملة.

### الـ Stack التقني
| الطبقة | التقنية |
|--------|---------|
| **الخادم** | Node.js + Express |
| **قاعدة البيانات** | SQLite (`data/database.sqlite`) + JSON files للـ cache |
| **الواجهة** | HTML5 + CSS3 (Vanilla) + JavaScript (Vanilla) |
| **الأمان** | Helmet.js + CORS |
| **الجدولة** | node-cron (جمع وتحليل كل 30 دقيقة) |
| **النشر** | GitHub → HostingGuru.io (deploy تلقائي) |

### ملاحظات وقيود
- الموقع بالكامل **RTL بالعربية**
- **لا توجد** واجهة MVC (React/Vue/Angular) — كل شيء Vanilla JS
- قاعدة البيانات **SQLite** — غير مناسبة للتوسّع الأفقي
- الصور تُخزّن كـ **Base64** في قاعدة البيانات (لا يوجد نظام ملفات حقيقي)
- بيئة التطوير: **Windows** (المسارات بـ `path.join`)
- بيئة الإنتاج: **Linux (HostingGuru.io)** — CRLF → LF تلقائي
